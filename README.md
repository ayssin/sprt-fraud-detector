\# SPRT Fraud Detector



Sequential Probability Ratio Test (SPRT) для детекции мошеннических транзакций на датасете IEEE-CIS Fraud Detection.



\## Структура проекта



\- `data/` — сырые и обработанные данные

\- `sql/` — SQL-схемы, функции, триггеры

\- `notebooks/` — Jupyter ноутбуки для анализа и визуализации

\- `scripts/` — Python скрипты для симуляции и интеграции с БД

\- `reports/` — графики, таблицы, финальные отчёты

\- `simulation/` — симулятор потока транзакций
